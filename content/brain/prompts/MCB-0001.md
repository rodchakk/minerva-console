# MCB-0001 — Brain Placement & Architecture Recon

## Mission

Reconnaissance mission. Inspect Minerva Console and recommend whether Brain should live inside the repo, in a separate repo, hybrid, or as a Git vault.

## Objective

- Analyze the existing Minerva Console codebase structure
- Evaluate placement options for the Brain intelligence layer
- Recommend one primary path with exact file paths
- Consider auth, routing, and deployment constraints

## Outcome

Recommended that Brain live inside Minerva Console under `app/(console)/brain/**` with feature code under `features/brain/**` and content under `content/brain/**`.
