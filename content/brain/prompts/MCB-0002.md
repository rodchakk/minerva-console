# MCB-0002 — Create Brain Harness and Static Shell

## Mission

Build mission. Scaffold the Brain content structure, harness files, JSON registries, feature module, and app routes. Reorganize the sidebar so Brain appears as a top-level section.

## Objective

- Create `content/brain/` directory structure with harness documents
- Create JSON registries for projects, decisions, prompts, agents, and inbox
- Create `features/brain/` module with types, content loader, and components
- Create `app/(console)/brain/**` routes
- Add Brain to the console sidebar navigation
- Add CI guardrails to enforce Brain boundaries

## Outcome

Brain v0 static shell is live with:
- 10 harness documents
- 5 JSON registries with seeded entries
- Feature module with typed content loader
- 6 routes (overview + 5 registry list pages)
- CI guardrails enforcing import boundaries
